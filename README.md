# J&A Little Egypt — Frontend

A Vite + React + TypeScript single-page site for the J&A Little Egypt restaurant. The design is text-forward with Egyptian/Mediterranean motifs and icon-based menu cards (no photos yet).

## Scripts

- `npm install` — install dependencies
- `npm run dev` — start dev server
- `npm run build` — type-check and build for production
- `npm run preview` — preview the production build
- `npm run lint` — run ESLint

## Notes

- Menu items use icons and textures instead of photos; swap in real imagery later if desired.
- Colors, typography, and layout tokens live in `src/styles.css`.
- Content data (menu, services, testimonials, contact) is defined in `src/App.tsx` for easy editing.
